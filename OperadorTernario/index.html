Ejercicios con Indicaciones (Usa el Operador Ternario)

Ejercicio 1:

Verifica si un número es positivo, negativo o cero. Usa un operador ternario para verificar si es mayor que 0, igual a 0, o menor que 0.

Indicaciones:

  • Si el número es mayor que 0, debería imprimir "Es positivo".
  • Si el número es igual a 0, debería imprimir "Es cero".
  • Si el número es menor que 0, debería imprimir "Es negativo".

Ejercicio 2:

Crea una función que reciba una calificación (número) y determine si la persona ha aprobado o reprobado.

Indicaciones:

  • Si la calificación es mayor o igual a 6, la persona ha aprobado.
  • Si la calificación es menor a 6, la persona ha reprobado.

Ejercicio 3:

Determina si un año es bisiesto o no. Usa el operador ternario.

Indicaciones:

  • Un año es bisiesto si es divisible entre 4, pero no divisible entre 100, a menos que también sea divisible entre 400.
  • Si el año es bisiesto, imprime "Es bisiesto".
  • Si no lo es, imprime "No es bisiesto".
