Ejercicio de la lección

Vamos a determinar si un usuario puede obtener una licencia de conducir.

  1. Crea una constante edadUsuario y asígnale el valor 20.
  2. Crea una constante pasoExamenVision y asígnale el valor true.
  3. Crea una variable puedeObtenerLicencia.
  4. En esa variable, guarda el resultado de verificar si edadUsuario es mayor o igual a 18 Y si pasoExamenVision es true. (Necesitarás un operador de comparación y uno lógico).
  5. Muestra el valor de puedeObtenerLicencia en la consola. El resultado debería ser true.

Ejercicio 1: Calculadora de Propinas (Básico)

El objetivo aquí es usar operadores aritméticos y de asignación simples.

Indicaciones:

  1. Crea una variable costoTotalCuenta y asígnale el valor 250.
  2. Crea una constante porcentajePropina y asígnale el valor 0.15 (que es el 15%).
  3. Calcula el montoPropina multiplicando el costo de la cuenta por el porcentaje de propina.
  4. Calcula el pagoFinal sumando el costoTotalCuenta y el montoPropina.
  5. Muestra en la consola un mensaje claro, por ejemplo: "El costo total es: [costoTotalCuenta], la propina es: [montoPropina], el pago final es: [pagoFinal]".

Ejemplo de salida esperada:

El costo total es: 250, la propina es: 37.5, el pago final es: 287.5

Ejercicio 2: Verificador de Acceso a un Evento (Intermedio)

Este ejercicio se enfoca en los operadores de comparación y lógicos (&&).

Indicaciones:

  1. Crea una constante edadMinima con el valor de 18.
  2. Crea una variable edadUsuario con un valor, por ejemplo, 25.
  3. Crea una constante tieneBoleto con el valor true.
  4. Crea una variable puedeIngresar que almacene el resultado de verificar si edadUsuario es mayor o igual a edadMinima Y si tieneBoleto es true.
  5. Muestra en la consola el valor de puedeIngresar.
  6. Prueba tu lógica: Cambia el valor de edadUsuario a 17 y vuelve a correr el código. Luego, cambia tieneBoleto a false y pruébalo de nuevo.

Ejemplo de salida esperada (con los valores iniciales):

¿Puede ingresar al evento?: true

Ejercicio 3: Lógica de Descuento para Clientes (Intermedio-Avanzado)

Aquí combinarás operadores lógicos && y || para una condición más compleja.

Indicaciones:

Un cliente obtiene un descuento si cumple una de estas dos condiciones:

  • Es un clienteFrecuente Y su montoCompra es mayor a 500.
  • O si tiene un cuponDeDescuento válido.
  1. Crea las siguientes variables:
    • esClienteFrecuente con el valor false.
    • montoCompra con el valor 600.
    • cuponDeDescuento con el valor true.
  2. Crea una variable aplicaDescuento. Su valor debe ser el resultado de la lógica descrita arriba. Pista: Necesitarás usar paréntesis () para agrupar la primera condición.
  3. Muestra en la consola un mensaje que diga: "El cliente aplica para descuento: [valor de aplicaDescuento]".
  4. Juega con los valores: Cambia los valores iniciales para probar todos los escenarios y asegurarte de que tu lógica es correcta. Por ejemplo, esClienteFrecuente = true y cuponDeDescuento = false.

Ejemplo de salida esperada (con los valores iniciales):

El cliente aplica para descuento: true

Ejercicio 4: Inversión de Estado (Avanzado)

Este ejercicio se enfoca en el operador de negación (!) y la reasignación con +=.

Indicaciones:

  1. Crea un objeto que represente una luz (un foco).
    2. const luz = { habitacion: "Sala de Estar", estaEncendida: false, brillo: 0 };
  2. Simula que enciendes la luz. Para ello, invierte el valor de estaEncendida usando el operador de negación !.
  3. Si la luz está encendida, su brillo debe ser 100. Si está apagada, 0. Usa un operador ternario para asignar el brillo basándote en el nuevo estado de estaEncendida.
  4. Muestra el objeto luz completo en la consola para ver el cambio.
  5. Ahora, simula que subes el brillo. Usa el operador de asignación += para sumarle 20 al brillo actual.
  6. Muestra un mensaje que diga "El nuevo brillo es de: [valor del brillo]".

Ejemplo de salida esperada:

{ habitacion: 'Sala de Estar', estaEncendida: true, brillo: 100 } El nuevo brillo es de: 120
